# Contributing to Healing Stones

Thank you for your interest in contributing to the Healing Stones project! We welcome contributions that help improve the reconstruction of cultural heritage artifacts using AI.

## How to Contribute

### 1. Reporting Bugs
- Use the GitHub Issues tracker to report bugs.
- Provide a clear description of the issue and steps to reproduce it.

### 2. Feature Requests
- If you have ideas for new features, feel free to open an issue for discussion.

### 3. Pull Requests
- Fork the repository and create a new branch for your changes.
- Ensure your code follows the existing style and is well-documented.
- Include tests for any new functionality.
- Submit a PR with a clear description of the changes.

## Coding Standards
- Use clear and descriptive variable names.
- Follow PEP 8 guidelines for Python code.
- Ensure all scripts can be run non-interactively as per project requirements.

## Contact
For any questions, reach out to the project mentors at human-ai@cern.ch.
